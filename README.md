# PruebaAutomatizacion
Prueba de automatización para web tipo marketplace. 
1. Flujo de búsqueda 
2. Flujo registro

Se utiliza :
-Eclipse IDE (Java)
- Gherkin
-Cucumber
-Selenium
-Junit

Para probar, debe bajar el repositorio y abrirlo desde su IDE. Para correr cada flujo se debe hacer desde los archivos "runner".
